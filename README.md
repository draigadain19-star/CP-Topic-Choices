# CP Topic Matcher Site

A simple React + Vite site for your World History CP topic-matching survey.

## Run locally
1. Open a terminal in this folder
2. Install packages:
   npm install
3. Start the dev server:
   npm run dev

## Build for production
npm run build

## Deploy for free
Recommended:
- Push this folder to a GitHub repository
- Import the repo into Vercel
- Deploy on the free Hobby plan

## Files
- `src/App.jsx` contains the survey questions, scoring logic, and result screen
- `src/styles.css` contains the styling
